# Saboratarekareka

Стартовый репозиторий для тестового задания: квиз-воронка подписки программ Екатерины Усмановой внутри Chatium.

## Что внутри

- `AGENTS.md` — главные инструкции для AI-агента.
- `docs/` — ограничения, архитектура, план реализации, логика результата, пейволл и чек-лист.
- `data/quiz_screen_map.json` — карта 24 экранов из ТЗ.
- `src/` — стартовые TypeScript-типы и функции для результата.
- `.github/copilot-instructions.md` — краткие инструкции для GitHub Copilot/Codex-подобных агентов.

## Цель

Собрать мобильную квиз-воронку в Chatium: вопросы, персональный результат, лид-гейт, сохранение данных, UTM, расчёт калорий/КБЖУ/ИМТ, пейволл и эмбед формы ГетКурса.

## Быстрый старт локально

```bash
npm install
npm run typecheck
```

Финальный код переносится в Chatium. Локальная структура нужна для подготовки логики, типов, инструкций и агентной разработки.

## Создание GitHub-репозитория через CLI

```bash
bash scripts/create-github-repo.sh saboratarekareka
```

Репозиторий будет создан как приватный в аккаунте `SaborataRekaReka`, если GitHub CLI авторизован.
