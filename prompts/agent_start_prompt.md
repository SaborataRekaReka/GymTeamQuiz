# Стартовый промпт для AI-агента

Ты помогаешь собрать тестовое задание в Chatium: платную квиз-воронку для подписки программ Екатерины Усмановой.

Прочитай файлы:

- `AGENTS.md`
- `docs/00_project_brief.md`
- `docs/01_hard_constraints.md`
- `data/quiz_screen_map.json`
- `docs/02_architecture.md`
- `docs/03_implementation_plan.md`
- `docs/04_result_logic.md`
- `docs/05_paywall.md`
- `docs/06_quality_checklist.md`

После чтения действуй так:

1. Сначала собери минимальный рабочий каркас квиза на 24 экрана.
2. Не меняй порядок экранов, вопросы и варианты ответов.
3. Вынеси все экраны в конфиг.
4. Сделай универсальный рендер для разных типов экранов.
5. Реализуй сохранение состояния.
6. Реализуй лид-гейт с именем, почтой и согласием.
7. Реализуй расчёт результата.
8. Реализуй персональный результат.
9. Реализуй пейволл.
10. Проверь результат по `docs/06_quality_checklist.md`.

Главный критерий: рабочая мобильная воронка без тупиков, с сохранением данных и персонализацией результата.

Не уходи в сложный визуал до того, как будет готов полный сквозной проход.
