# 06. Чек-лист качества и приёмки

Используй этот файл перед сдачей.

## Сквозной сценарий

- [ ] Квиз открывается на телефоне.
- [ ] Первый экран загружается быстро.
- [ ] Все 24 экрана доступны по порядку.
- [ ] Нет тупиков.
- [ ] Все кнопки работают.
- [ ] Все обязательные поля валидируются.
- [ ] Пейволл открывается после результата.

## Ответы

- [ ] Одиночный выбор работает.
- [ ] Множественный выбор работает.
- [ ] Вес сохраняется.
- [ ] Рост сохраняется.
- [ ] Целевой вес сохраняется.
- [ ] Имя сохраняется.
- [ ] Почта сохраняется.
- [ ] Чекбокс согласия обязателен.
- [ ] UTM-метки сохраняются.

## Персонализация

- [ ] Имя подставляется в результат.
- [ ] Текущий вес подставляется в результат.
- [ ] Целевой вес подставляется в результат.
- [ ] Цель подставляется в сводку.
- [ ] Зоны подставляются в сводку.
- [ ] Формат тренировок подставляется в сводку.
- [ ] Тип результата присваивается.
- [ ] Подборка программ зависит от ответов.

## Расчёты

- [ ] Норма калорий считается.
- [ ] КБЖУ считается.
- [ ] ИМТ считается.
- [ ] Шкала индекса массы тела отображается.
- [ ] Срок пути к цели отображается.
- [ ] Допущения по коэффициентам отмечены в сдаче.

## Лид-гейт

- [ ] Почта запрашивается только на экране 21.
- [ ] Нет обещания отправить результат на почту.
- [ ] Юридические ссылки у чекбокса работают.
- [ ] Без галочки нельзя пройти дальше.
- [ ] Лид сохраняется в базу.

## Пейволл

- [ ] Есть оффер от результата человека.
- [ ] Есть социальные доказательства.
- [ ] Есть блок «до и после».
- [ ] Есть все три тарифа.
- [ ] Есть текст про автопродление.
- [ ] Есть состав подписки под цель.
- [ ] Есть гарантии и значки безопасности.
- [ ] Есть FAQ.
- [ ] Есть юридический футер.
- [ ] Встроена форма ГетКурса или стоит помеченный TODO.

## Текстовые правила

- [ ] В пользовательском тексте кавычки только «ёлочки».
- [ ] Нет обычных кавычек.
- [ ] Нет длинных тире как разделителей.
- [ ] Нет стрелок.
- [ ] Нет английских слов в пользовательских текстах.
- [ ] Везде обращение на «вы».
- [ ] Нет обесценивания возраста и тела.
- [ ] Нет фраз про «осталось N мест».

## Визуал

- [ ] Фон светлый.
- [ ] Не больше двух акцентных цветов.
- [ ] Текст читается на телефоне.
- [ ] Кнопки заметные.
- [ ] Изображения сжаты.
- [ ] Нет тяжёлых видео.
- [ ] Нет мигающих и прыгающих элементов.
- [ ] Нет тёмной темы.

## Техническая проверка

- [ ] Нет ошибок в консоли, если доступна консоль.
- [ ] Нет зависаний при переходах.
- [ ] Данные не теряются при прохождении.
- [ ] Повторное прохождение создаёт новую запись или корректно обновляет текущую.
- [ ] Рабочая версия протестирована минимум на 3 сценариях.
